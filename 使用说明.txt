使用方法：

1. 把需要处理的照片放进“输入图片”文件夹
2. 照片文件名必须是：分厂-姓名-YYYYMM
   例如：八分厂-范鸿勇-202603.jpg
3. Mac 双击“运行月度之星海报.command”
4. Windows 双击“运行月度之星海报-Windows.bat”
5. 生成好的海报会出现在“输出海报”文件夹

说明：

- 不会覆盖原始照片
- 输出文件名格式：原文件名-月度之星.png
- 使用模板文件：月度之星个人海报-素材模板4.png
- Windows 电脑需要先安装 Python 3

如果要生成 Windows 的 EXE：

1. 把整个“月度之星批处理工具”文件夹发到 Windows 电脑
2. 在 Windows 上双击“在Windows上打包EXE.bat”
3. 打包完成后，EXE 会出现在 dist 文件夹
